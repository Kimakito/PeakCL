Carte carré-arrondi, le conteneur de base du système.

```jsx
<Card kicker="Palier 01" title="On fait le point">Un appel de 30 min pour cadrer tes besoins.</Card>
<Card tone="lavande" title="Identité visuelle">Logo, couleurs, typographies.</Card>
```

tone applique un dégradé de famille (lavande/violet ⇒ texte blanc). hover=false pour désactiver la remontée.
